Display thumbnails of user profiles registered on the site with options for random order and quantity limit.

== Description ==

This plugin allows you to display thumbnails of user profiles registered on your site using a shortcode. You can configure the border color, border size, thumbnail size, and activate the random mode to display different profiles on each page load.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/profile-images` directory, or install the plugin directly through the WordPress plugins screen.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the `[profile_images]` shortcode to display user profile thumbnails.

== Frequently Asked Questions ==

= How do I use the shortcode? =

You can use the `[profile_images]` shortcode in any page or post to display user profile thumbnails.

= How do I configure the plugin? =

You can configure the plugin through the settings page in 'Settings > Profile Images' in the WordPress admin panel.
